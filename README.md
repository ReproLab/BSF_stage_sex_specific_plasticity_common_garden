# BSF_stage_sex_specific_plasticity_common_garden
Data and R scripts for a common garden experiment testing genotype-, stage-, and sex-specific phenotypic plasticity across metamorphosis in the black soldier fly (Hermetia illucens).

All statistical analyses and figures reported in the manuscript can be reproduced using the R scripts provided. Each script specifies the required input dataset to load from the `data/` directory. Please follow the script order and load the indicated datasets to generate the corresponding outputs.
